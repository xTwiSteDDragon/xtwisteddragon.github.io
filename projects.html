Anti-Phishing Discord Bot - Spring 2026

I completed this project as a part of a the CSC450: Network Security class at TCNJ. I was assigned an open-ended project to either learn or create something impactful, and seeing as I have friends who got their accounts hacked on Discord by bots, I decided to use bots on chat servers themselves as a way to prevent bot-based phishing attacks.

The bot became known as KingPhisher, and is built entirely in Python, using Discord's bot API. It utilizes the SQL-Lite Python library to create and manage an SQL database storing links and domains that should be blocked or kept on a chat server when a user sends them. If a new link is dicovered by the bot, it verifies that the user sending the link is a human and therefore keeps the link the server. If this verification is not completed, the message containing the link is deleted, and the same link is blacklisted.

My team and I also added in functionality to flexibly manage the bot's behavior by a chat server admin, allowing entire domains to be covered by a single whitelist addition instead of redundant links filling the whitelist. Ultimately, the entire bot is meant to prevent compromised accounts to send malicious links for phishing, and infecting other accounts.
